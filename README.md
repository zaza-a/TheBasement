# I used to pray for times like these.\
This project is a fully responsive Escape Room web experience built by a high school student. It includes interactive puzzles, custom animations, and modal-based clues.
I used parameterized functions, multiple onclick handlers, and clean, reusable JavaScript to manage different interactions throughout the game.
The goal was to create an engaging, mobile-friendly challenge while improving my front-end skills in HTML, CSS, and JavaScript.
